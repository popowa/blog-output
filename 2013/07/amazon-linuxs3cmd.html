Amazon Linuxにはs3cmdが入っていると聞いていたけど入っていなかったので、入れてみた。

wget http://s3tools.org/repo/RHEL_6/s3tools.repo \~/
sudo cp s3tools.repo /etc/yum.repos.d/
sudo yum -y install s3cmd
